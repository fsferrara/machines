# Initial Setup for Windows

This directory contains scripts and instructions for setting up a Windows environment for development and productivity.

## Contents

- `execution-policy-bypass.bat`: this needs to be executed before running powershell ps1 script.
- `install_winget_tools.ps1`: Installs essential tools and technologies using Windows Package Manager (winget).
- `wsl`: Sets up Windows Subsystem for Linux (WSL)

## Notes

- Review the scripts before running them to customize the setup as needed.
